Merci d'avoir téléchargé Isilex.

Démarrer:
---------

Si vous disposez déjà d'une version de Java installée sur votre ordinateur, il suffit de
double cliquer sur Isilex.jar. 

Sinon, il existe une version d'IsilexLight avec Java embarqué. Pour démarrer, vous 
trouverez un fichier script de lancement dans le répertoire d'Isilex.

Si votre ordinateur émet des avertissements de sécurité,
faites confiance à la suite Isilex.

Une fois démarré, vous devriez voir apparaître une icône verte près de votre horloge.

Cela signifie qu'IsilexLight est démarré et prêt à recevoir et servir des données.

Si vous travaillez derrière un firewall, n'oubliez pas d'ouvrir les ports 8984 et 8985 
pour UDP/TCP dans les deux sens.

Travailler:
-----------

Votre suite est accessible à l'adresse http://localhost:8984.

D'autres ordinateurs de votre réseau peuvent immédiatement accéder aux données que vous 
saisissez sur votre adresse locale. Si votre ordinateur est connecté à Internet et que
vous disposez d'une adresse Web accessible, votre Isilex est accessible à l'adresse:

- http://mondomaine.ip:8984

D'autres utilisateurs peuvent se connecter à cette même adresse pour s'enregistrer et 
collaborer à votre projet.
